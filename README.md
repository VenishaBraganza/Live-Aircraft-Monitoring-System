# ADS-B Aircraft Surveillance and Tracking System using RTL-SDR

## Overview

A real-time aircraft monitoring and surveillance system built using RTL-SDR, RTL1090, Virtual Radar Server, and a 1090 MHz high-gain antenna. The system receives ADS-B broadcasts from nearby aircraft and displays live flight information including aircraft position, altitude, speed, heading, callsign, and route details.

The setup provides aircraft tracking coverage of up to 500 km depending on antenna placement, terrain, and atmospheric conditions.

## Features

* Real-time aircraft tracking
* Live ADS-B signal reception
* Aircraft position monitoring
* Altitude and speed tracking
* Heading and route visualization
* Callsign and aircraft identification
* Interactive radar display
* Long-range reception up to 500 km
* Continuous air traffic monitoring
